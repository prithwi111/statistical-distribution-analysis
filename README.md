# Statistical Distribution Analysis Project

## Overview
This project performs exploratory data analysis (EDA) on restaurant transaction data to understand distribution patterns, skewness, spread, and outliers.

## Objectives
- Visualize distributions using Histogram and KDE
- Detect outliers using Boxplot
- Compare distributions across groups (Smoker vs Non-Smoker)
- Provide business interpretation

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
